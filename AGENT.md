# AGENT.md

## 프로젝트 목표

미국 주식 1분봉 데이터를 이용해 **초단타 매수·매도 타이밍을 판단하는 AI 모델**을 개발한다.

모델은 가격 급등 직전의 진입 기회와 진입 이후의 청산 시점을 판단하는 것을 목표로 한다.
구체적인 데이터 전처리, 특징, 라벨, 모델 구조, 평가 방법은 이후 단계별로 검토하여 이
문서에 추가한다.

## 프로젝트 경로

모든 상대 경로는 현재 실행 위치가 아니라 **프로젝트 루트 기준**으로 해석한다.

```text
데이터:  ../../data/stock_data
모델:    ../../model/
결과:    ../../results/
```

- 원본 데이터는 데이터 경로에서 읽는다.
- 학습된 모델과 모델 관련 artifact는 모델 경로에 저장한다.
- 전처리 결과, 학습 지표, 예측값, 백테스트 결과는 결과 경로에 저장한다.
- 원본 데이터와 생성 artifact를 섞지 않는다.

## 현재 진행 원칙

- 기존 실험 전략을 전제로 하지 않고 처음부터 설계한다.
- 한 단계씩 검토한 뒤 확정된 내용만 이 문서에 추가한다.
- 실험 결과를 확인하기 전에 목표나 평가 기준을 임의로 변경하지 않는다.
